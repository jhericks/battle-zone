# Product Overview

BattleZone-style tank combat game where the player controls a tank using dual-tread controls to battle enemy tanks in a vector-graphics arena.

## Core Gameplay
- Destroy enemy tanks to earn points (1 point per kill)
- Survive as long as possible - game ends when player is hit
- Navigate around stationary obstacles that block movement and shells
- Enemy tanks increase in speed with each kill, ramping up difficulty

## Visual Style
- Retro vector graphics aesthetic with purple (#a855f7) wireframe rendering
- Black background with purple line-based graphics
- Kiro logo used as shell sprite
- Minimalist, arcade-inspired UI

## Game States
- **Start**: Title screen with controls explanation
- **Playing**: Active combat gameplay
- **Game Over**: Final score display with restart option
